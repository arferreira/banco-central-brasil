## Sobre

Este pacote em Go oferece acesso às taxas oficiais diárias de **Inflação**, **Selic**, **Poupança**, **Dólar**, **Dólar PTAX**, **Euro** e **Euro PTAX** por meio do site do Banco Central do Brasil (http://www.bcb.gov.br).
